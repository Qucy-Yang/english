\section{小学作文的积累}

  \subsection{讲述事情}
学会使用顺序描述词:

\begin{itemize}
    \item First
    \item Then
    \item Next
    \item After that
    \item Finally
\end{itemize}

如果描述好，那么需要会写下面几个：

\begin{itemize}
    \item good
    \item perfect
    \item terrific
    \item nice
    \item excellent
\end{itemize}

注意surprise这个词，Qucy在练习中没有正确书写出来
